has been found in the store. You can now use it in other function calls, or export it.

Using the stored object in a function call

To use the stored object in a function call, simply pass in the key as an argument. For example to call the 'class' function: 

 /R/call/base/class/json?x=/TEMPLATEKEY

Exporting the object

To export the object and download it in a client or browser, simply add an output type to the current url.
For example, choose one of the output types below to export it.

Export general R object

These can be used for any type of R object

  • /encode -- encodes to opencpu encoding (json)
  • /ascii -- export to dput ascii
  • /rds -- export as RDS
  • /rda -- export as RData

Render Plots

Use these only if the object is a plot/graph

  • /png -- renders a plot to png
  • /pdf -- renders a plot to pdf
  • /svg -- render to svg

Export basic R data structure

Use the simple json output only for basic R datastructures, such as vectors, lists and dataframes.

  • /json -- encode to simple json

Export a dataframe

Use these outputs only if the object is a dataframe.

  • /csv -- export as CSV
  • /csv -- export as CSV2 (European)
  • /table -- export as Tab Delimited

Export a raw vector / file

Use the /bin output only if the object is a raw vector. Files that were saved with /save are also saved as raw.

  • /bin -- sends a raw binary
